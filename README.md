# devdevedv

Μοντέρνο portfolio website για την ομάδα DevTaskHub - Full-Stack Developers στη Θεσσαλονίκη.

## 📋 Περιγραφή

Η DevTaskHub είναι μια ομάδα developers που προσφέρει υπηρεσίες ανάπτυξης λογισμικού, web development, mobile applications, AI integrations και άλλες τεχνολογικές λύσεις. Αυτό το website είναι το επίσημο portfolio και presentation site της εταιρείας.

## ✨ Χαρακτηριστικά

- 🎨 **Modern UI/UX** - Σχεδιασμένο με Tailwind CSS και Framer Motion
- 🌍 **Δίγλωσσο** - Υποστήριξη Ελληνικών και Αγγλικά
- 📱 **Responsive Design** - Πλήρως responsive για όλες τις συσκευές
- ⚡ **Fast & Optimized** - Χτισμένο με Vite για απόδοση
- 🎭 **Animations** - Smooth animations με Framer Motion και GSAP
- 🔍 **SEO Optimized** - React Helmet για SEO optimization
- 🎯 **TypeScript** - Type-safe development

## 🛠️ Τεχνολογίες

- **React 18** - UI Framework
- **TypeScript** - Type Safety
- **Vite** - Build Tool
- **React Router** - Routing
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **GSAP** - Advanced Animations
- **React Helmet** - SEO
- **React Icons** - Icons
- **AOS** - Scroll Animations
- **Lottie React** - Lottie Animations

## 🚀 Εγκατάσταση

```bash
# Clone το repository
git clone https://github.com/sio2000/devdevedv.git

# Μεταβείτε στον φάκελο
cd devdevedv

# Εγκαταστήστε τα dependencies
npm install
```

## 📝 Scripts

```bash
# Development server
npm run dev

# Build για production
npm run build

# Preview production build
npm run preview

# Linting
npm run lint
```
